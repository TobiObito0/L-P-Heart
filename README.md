# L-P-Heart



Trước tiên, bạn hãy tải thư mục về máy tính và tiến hành giải nén.
Sau đó, mở file heart.html trước để chương trình hoạt động đúng.

First, please download the folder to your device and extract it.
Then, open the heart.html file first to run the project correctly.



Trong trường hợp file heart.html không mở được hoặc mở lên nhưng không hiển thị nội dung, bạn vui lòng đóng lại và mở file dự phòng backup_file.html để tiếp tục,

If the heart.html file cannot be opened or shows no content, please close
it and open the backup file backup_file.html instead.
